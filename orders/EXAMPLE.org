#+TITLE: Kolabi Org Strategy Table CLI Examples

This file shows the active strategy grammar used by =kolabi.bot=. Strategy files are Org tables, and =run-once= uses the same typed value syntax for one pair.

* Typed Value Rules

- =qty= is required and starts with =A= or =%=, for example =A11=.
- =pGate= is a typed two-bound interval, for example =D1 2=, =D- +=, =%-1 1=, or =A79320 79340=.
- =hPrice= is a lazy typed head price, for example =D6=, =%0.2=, or =A79350=; it is computed only when =pGate= opens.
- =tPrice= is a typed tail distance, for example =D20=, =%1.2=, or =A0.25=.
- =hDelta= starts with =D= or =%= when present and is only for =SL= or =LT= head trigger-to-limit offsets.
- =tDelta= starts with =D= when present.
- =tUblk= starts with =D= or =%= when present.
- =wUblk= is a plain number of minutes; blank defaults to 6.

Removed compatibility flags and old keyword spellings are intentionally absent: =--aType=, =-a=, =--quantity=, =-q=, =--tailPrice=, =-T=, =--pgate=, =--hprice=, =--tp=, and =--oDelta= are no longer accepted.

* One Pair Commands

#+begin_src bash
python -m kolabi.bot run-once \
  --symbol PI_XBTUSD \
  --environment demo \
  --name XSellTail \
  --tps_run 0 1440 \
  --tOut 60 \
  --side sell \
  --oType L \
  --hPrice D1 \
  --qty A1 \
  --tType S- \
  --pGate 'D1 2' \
  --tPrice %0.5 \
  --dry-run

python -m kolabi.bot run-once \
  --symbol PI_XBTUSD \
  --environment demo \
  --name XPct \
  --tps_run 0 60 \
  --side buy \
  --oType L \
  --hPrice %0.5 \
  --qty A1 \
  --tType S- \
  --pGate '%-1 1' \
  --tPrice %0.5 \
  --dry-run

python -m kolabi.bot run-once \
  --symbol PI_XBTUSD \
  --environment demo \
  --name XAbs \
  --tps_run 0 60 \
  --side sell \
  --oType L \
  --hPrice A79350 \
  --qty A1 \
  --tType S- \
  --pGate 'A79320 79340' \
  --tPrice D0.5 \
  --dry-run
#+end_src

| KRKF | PI_ADAUSD | KADA_SEL | 0 1440 | 1 | 6 |  | sell | L |  | A1 | S- |  | D- + | %0.20 | %0.10 |  |  | XBTC_SEL-head-filled |
| BINS | ADAUSDT | NADA_BUY | 0 1440 | 1 | 6 |  | buy | M |  | A3 | L! |  | D- + |  | D0 |  |  | KADA_SEL-tail-closed |
