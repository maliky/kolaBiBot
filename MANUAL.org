* Overview

- This file lists the main command-line entry points implemented in this repo.
- Default focus is Kraken Futures.
- Use =--environment demo= for the demo platform and =--environment live= for the live platform.
- Active trade entry points now live under =kolabi.bargain/=.
- If a command fails because of a symbol, first list instruments with:
  - =python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo instruments=
- Standard order intents currently covered by the Kraken contract:
  - =Limit=
  - =Market=
  - =StopLoss= or trigger-entry market
  - =StopLossLimit= or trigger-entry limit
  - =TakeProfit= or market-if-touched
  - =TakeProfitLimit= or limit-if-touched
  - =TrailingStop= native Kraken trailing stop
  - =TrailingStopLimit= native Kraken trailing stop limit
- In this repo, order placement is currently done through Kraken Futures REST.
- Websockets are currently used here for public market data and private account/order state, not for placing Futures orders.

* trade.cli

- Command: =python -m kolabi.bargain.cli=
- Role: direct Kraken Futures CLI for instruments, balance, position, order submission, amend, and cancel
- Global options:
  - =--symbol= : selected Kraken Futures product id, example: =PI_XBTUSD=
  - =--environment {demo,live}= : selected endpoint family

| Command          | Purpose                                | Main options                                          |
|------------------+----------------------------------------+-------------------------------------------------------|
| =balance=        | read available margin                  | none                                                  |
| =position=       | read current position                  | none                                                  |
| =instruments=    | list Futures instruments               | =--contains=                                          |
| =check-symbol=   | verify the selected product id         | none                                                  |
| =limit=          | send one limit order                   | =--side=, =--qty=, =--price=                          |
| =market=         | send one market order                  | =--side=, =--qty=                                     |
| =cancel=         | cancel one order                       | =--order-id=                                          |
| =amend=          | amend one order                        | =--order-id=, =--price=, =--qty=                      |
| =trailing=       | send native Kraken trailing stop       | =--side=, =--qty=, =--deviation=, =--unit=            |
| =trailing-limit= | send native Kraken trailing stop limit | =--side=, =--qty=, =--price=, =--deviation=, =--unit= |

Examples:

#+begin_src bash
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo balance
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo position
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo instruments --contains ADA
  python -m kolabi.bargain.cli --symbol PF_ADAUSD --environment demo check-symbol
  # orders
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo limit --side buy --qty 2 --price 70000
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo market --side buy --qty 1
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo cancel --order-id YOUR_ORDER_ID
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo amend --order-id YOUR_ORDER_ID --price 70100 --qty 3
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo trailing \
    --side buy --qty 1 --deviation 20 --unit PERCENT
  python -m kolabi.bargain.cli --symbol PI_XBTUSD --environment demo trailing-limit \
    --side buy --qty 1 --price 82000 --deviation 20 --unit PERCENT

* tree.kraken public db

- Command: =python -m kolabi.tree.kraken=
- Role: public Kraken Futures market-data DB service (orderbook snapshots and indicators)
- Data persisted: market snapshots, derived indicators, service health

| Subcommand | Purpose                                                    |
|------------+------------------------------------------------------------|
| =run=      | start continuous public websocket listener and DB writer   |
| =probe=    | run listener for bounded seconds, then print status        |
| =status=   | show public DB counts and stream status                    |

Shared options:

| Option                             | Meaning                                   |
|------------------------------------+-------------------------------------------|
| =--pair=                           | Futures product id, for example PI_XBTUSD |
| =--environment {demo,live}=        | endpoint family                           |
| =--db-url=                         | public SQLite DB location                 |
| =--private-db-url=                 | private DB location used for correlation  |
| =--ws-url=                         | override public websocket URL             |
| =--trace-ws=                       | print websocket payload traces            |
| =--trace-ws-format {compact,json}= | trace output format                       |
| =--snapshot-interval-seconds=      | snapshot write cadence                    |
| =--indicator-interval-seconds=     | indicator write cadence                   |
| =--log-interval-seconds=           | service heartbeat cadence                 |

Examples:

#+begin_src bash
python -m kolabi.tree.kraken run --environment demo --pair PI_XBTUSD
python -m kolabi.tree.kraken probe --environment demo --pair PI_XBTUSD --seconds 15 --trace-ws
python -m kolabi.tree.kraken status --environment demo --pair PI_XBTUSD
#+end_src
