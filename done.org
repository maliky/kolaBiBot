** DONE <2026-06-11 jeu. 06:30> typed TSV grammar
CLOSED: [2026-06-12 ven.]

- Active strategy rows use =exchg= and =qty= at the operator boundary.
- Empty TSV columns are ignored, so template spacing columns do not become strategy fields.
- Non-empty typed fields carry their own prefix: =D=, =%=, or =A= where the field supports it.
- Only empty typed fields receive implicit defaults. A value such as =6= in =oDelta= is rejected; write =D6= instead.

** DONE <2026-06-02 mar. 18:28> multi-platform route model
CLOSED: [2026-06-12 ven.]

- v1 originally targeted derivatives/perpetuals only; this is superseded by the multi-exchange route model that supports futures, spot, and margin lanes where the exchange adapter supports them.
- every platform should have the same DB-grounded architecture from day one
- Coinbase is still out of scope. Kraken, Binance, and BitMEX are in the shared route matrix.
- Platform-specific symbols be written directly in TSV files. We should not have canonical symbols a mapping table.  The reason is to be able to have one TSV table calling synbole for different plateform. In fact it is important that symbols for same instrument do not colide.
- Unsupported grammar should fail at parse/preflight time
- Superseded: the runtime now supports rows with different exchange/market/symbol routes in one TSV. Each route still needs its own fresh public/private DB feeder evidence.

** DONE Demo Cross-Exchange Chain MWE
CLOSED: [2026-06-12 ven.]

The demo strategy is stored in:

#+begin_example
orders/demo_cross_exchange_chain.org
#+end_example

Intent:
- =XBTC_SEL= on BitMEX futures =XBTUSD=: sell limit head =D6= USD above the BTC reference, then protect with a buy stop tail =D20= USD above the fill/reference.
- =KADA_SEL= on Kraken Futures ADA: released by =XBTC_SEL-head-filled=, sell limit head =%0.20= above the ADA reference, then protect with a buy stop tail =%0.10= above the fill/reference.
- =NADA_BUY= on Binance spot =ADAUSDT=: released by =KADA_SEL-tail-closed=, buy a fixed ADA quantity from the TSV at market, then emit an =L!= zero-distance limit tail intended to cancel immediately instead of leaving a standing order.

Before running, probe the exact demo symbols:

#+begin_src bash
python -m kolabi.bargain.cli --exchange bitmex --market-type futures --environment demo --symbol XBTUSD check-symbol
python -m kolabi.bargain.cli --exchange kraken --market-type futures --environment demo instruments --contains ADA
python -m kolabi.bargain.cli --exchange kraken --market-type futures --environment demo --symbol PI_ADAUSD check-symbol
python -m kolabi.bargain.cli --exchange binance --market-type spot --environment demo --symbol ADAUSDT check-symbol
#+end_src
